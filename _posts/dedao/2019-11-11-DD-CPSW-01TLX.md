## 1 潜意识

```
意识即防御
```

用户会在当时的场景下做出==自认为==“正确的话”，但这并不代表用户的真实选择。


**角色建立步骤**

```
1. 确定目标
2. 建立恐惧的世界
3. 放入清晰的标的
4. 给予红利，然后让他产生路径依赖
5. 重复
```

> 人只有在很高的压力下才会按照角色行事。

自私是自我的一部分。

创造力是自我的延伸，控制力是自律的延伸。


```
靠愉悦驱动的人，往往自我。
靠恐惧驱动的人，往往自律。
```
